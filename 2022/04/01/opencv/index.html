1.3图像旋转

在OpenCV中图像旋转首先根据旋转角度和旋转中心获取旋转矩阵，然后根据旋转矩阵进行变换，即可实现任意角度和任意中心的旋转效果。

# 返回一个矩阵
cv.getRotationMatrix2D(center, angle, scale)

M = cv.getRotationMatrix2D((cols/2,rows/2),90,1)
dst = cv.warpAffine(img,M,(cols,rows))
'''
center：旋转中心
angle：旋转角度
scale：缩放比例
'''

3、图像平滑

3.1图像噪声

椒盐噪声：一种随机出现的白点或者黑点，可能是亮的区域有黑色像素或是在暗的区域有白色像素（或是两者皆有）

高斯噪声：噪声密度函数服从高斯分布

3.2图像平滑

图像平滑从信号处理的角度看就是去除其中的高频信息，保留低频信息

低通滤波可以去除图像中的噪声，对图像进行平滑

均值滤波:
它只是用卷积框覆盖区域所有像素的平均值来代替中心元素。
去噪的同时去除了很多细节部分，将图像变得模糊。

cv.blur(img, ksize, anchor, borderType)
# ksize：卷积核的大小
# anchor：默认值 (-1,-1) ，表示核中心
# borderType：边界类型

高斯滤波:

将”中心点”作为原点，其他点按照其在正态曲线上的位置，分配权重，就可以得到一个加权平均值

cv.GaussianBlur(img,ksize,sigmaX,sigmay,borderType)
# ksize:高斯卷积核的大小，注意 ： 卷积核的宽度和高度都应为奇数，且可以不同
# sigmaX: 水平方向的标准差
# sigmaY: 垂直方向的标准差，默认值为0，表示与sigmaX相同
# borderType:填充边界类型

中值滤波：

用像素点邻域灰度值的中值来代替该像素点的灰度值。

对椒盐噪声来说尤其有用，因为它不依赖于邻域内那些与典型值差别很大的值。

cv.medianBlur(src, ksize)

4.2 直方图均衡化

这种方法提高图像整体的对比度，特别是有用数据的像素值分布比较接近时，可以提高骨架结构的显示，另外在曝光过度或不足的图像中可以更好的突出细节。

自适应的直方图均衡化：整幅图像会被分成很多小块，这些小块被称为“tiles”（在 OpenCV 中 tiles 的 大小默认是 8x8），然后再对每一个小块分别进行直方图均衡化

dst = cv.equalizeHist(img)

# 自适应的直方图均衡化
clahe = cv.createCLAHE(clipLimit, tileGridSize)
res = clahe.apply(img)
# clipLimit: 对比度限制，默认是40
# tileGridSize: 分块的大小，默认为8*8

5、边缘检测

标识数字图像中亮度变化明显的点

基于搜索：通过寻找图像一阶导数中的最大值来检测边界，然后利用计算结果估计边缘的局部方向，通常采用梯度的方向，并利用此方向找到局部梯度模的最大值，代表算法是Sobel算子和Scharr算子。

基于零穿越：通过寻找图像二阶导数零穿越来寻找边界，代表算法是Laplacian算子。

5.1 Sobel检测算子 / Scharr算子

Sobel算子：高斯平滑与微分操作的结合体，所以其抗噪声能力很强，用途较多。尤其是效率要求较高，而对细纹理不太关心的时候。

Sobel算子是在两个方向计算的, cv2.addWeighted( )函数将其组合

Scharr算子：但该函数仅作用于大小为3的内核。该函数的运算与Sobel函数一样快，但结果却更加精确

对灰度渐变和噪声较多的处理效果比较好

Sobel_x_or_y = cv.Sobel(img, ddepth, dx, dy, dst, ksize, scale, delta, borderType)
Scale_abs = cv.convertScaleAbs(x)  # 格式转换函数
result = cv.addWeighted(Sobel_x, alpha, Sobel_y, beta) # 图像混合

# ddepth: 图像的深度
# dx和dy: 指求导的阶数，0表示这个方向上没有求导，取值为0、1。
# ksize: 是Sobel算子的大小，即卷积核的大小，必须为奇数1、3、5、7，默认为3。
#   注意：如果ksize=-1，就演变成为3x3的Scharr算子。
# scale：缩放导数的比例常数，默认情况为没有伸缩系数。
# borderType：图像边界的模式，默认值为cv2.BORDER_DEFAULT。

5.2 Laplacian算子

Laplacian是利用二阶导数来检测边缘

造成边缘丢失

laplacian = cv.Laplacian(img, ddepth[, dst[, ksize[, scale[, delta[, borderType]]]]])
# Ddepth: 图像的深度，-1表示采用的是原图像相同的深度，目标图像的深度必须大于等于原图像的深度；
# ksize：算子的大小，即卷积核的大小，必须为1,3,5,7

5.3 Canny边缘检测

原理：噪声去除(高斯滤波器)+计算图像梯度+非极大值抑制+滞后阈值（当图像的灰度梯度高于 maxVal 时被认为是真的边界， 低于 minVal 的边界会被抛弃）

不易受噪声干扰，容易检测出真正的弱边缘

canny = cv.Canny(img, threshold1, threshold2)
# img:灰度图
# threshold1: minval，较小的阈值将间断的边缘连接起来
# threshold2: maxval，较大的阈值检测图像中明显的边缘

5.4 代码展示

import numpy as np
import cv2 as cv
import os
import d2data
import matplotlib.pyplot as plt
import matplotlib

img1 = cv.imdecode(np.fromfile(r"./海贼图片/1.jpg",dtype=np.uint8),1)

# 1、Sobel算子
x = cv.Sobel(img1, cv.CV_16S, 1, 0)   # 计算Sobel卷积结果
y = cv.Sobel(img1, cv.CV_16S, 0, 1)
Scale_absX = cv.convertScaleAbs(x)    # convert转换 scale缩放
Scale_absY = cv.convertScaleAbs(y)
result1 = cv.addWeighted(Scale_absX, 0.5, Scale_absY, 0.5, 0)   # 结果合成

# 2、Scharr
x = cv.Sobel(img1, cv.CV_16S, 1, 0, ksize = -1)
y = cv.Sobel(img1, cv.CV_16S, 0, 1, ksize = -1)
Scale_absX = cv.convertScaleAbs(x)    # convert转换 scale缩放
Scale_absY = cv.convertScaleAbs(y)
result2 = cv.addWeighted(Scale_absX, 0.5, Scale_absY, 0.5, 0)

# 3、Laplacian算子
result3 = cv.Laplacian(img1,cv.CV_16S)
result3 = cv.convertScaleAbs(result3)

# 4、Canny
img2 = cv.imdecode(np.fromfile(r"./海贼图片/1.jpg",dtype=np.uint8),0)  # 灰度图
lowThreshold = 0
max_lowThreshold = 100
result4 = cv.Canny(img2, lowThreshold, max_lowThreshold)

# 图像展示
plt.rcParams['font.sans-serif']=['SimHei']
plt.rcParams['axes.unicode_minus'] = False
fig,axes = plt.subplots(nrows=1,ncols=5,figsize=(50,35),dpi=100)
axes[0].imshow(img1[:,:,::-1])
axes[0].set_title("原图")
axes[1].imshow(result1[:,:,::-1])
axes[1].set_title("Sobel")
axes[2].imshow(result2[:,:,::-1])
axes[2].set_title("Scharr")
axes[3].imshow(result3[:,:,::-1])
axes[3].set_title("Laplacian")
axes[4].imshow(result4,cmap = plt.cm.gray)
axes[4].set_title("Canny")
plt.show()
