多元统计 - 画图

发表于 2022-04-22 更新于 2022-05-08 分类于 多元统计 阅读次数：

数据下载：https://github.com/tangbingqing/multivariate-statistics-data

轮廓图和调和曲线图有着相近的功能，轮廓图简单却现得粗糙，调和曲线图公式复杂却十分精细。

均可用于聚类分析：同类曲线聚在一起，不同类不同束

一、轮廓图

在二维直角坐标系中，将n次观测值可画出n条折线构成轮廓图，其中横坐标表示各个变量，纵坐标表示与变量取值成正比的数值

data = pd.read_csv(r"iris.data")   # 尾鸢花数据
plt.figure()
pd.plotting.parallel_coordinates(data,'species')
plt.show()

二、调和曲线图

from pandas.plotting import andrews_curves

data = pd.read_csv(r"iris.data")
plt.figure()
andrews_curves(data,'species')
plt.show()

三、雷达图

matplotlib.rcParams['font.family']='SimHei'
matplotlib.rcParams['font.sans-serif']=['SimHei']
labels = np.array(['综合','第一周','第二周','第三周','第四周','第五周','第六周'])
score = np.array([88.7,85,90,95,70,96])

nAttr = 6  # 共6组
angles=np.linspace(0,2*np.pi,nAttr,endpoint=False)

Python=np.concatenate((score,[score[0]]))  # 闭合
angles=np.concatenate((angles,[angles[0]]))  #闭合

fig=plt.figure(facecolor="white")   # 背景颜色，白色

plt.subplot(111,polar=True)
plt.plot(angles,Python,'bo-',color='g',linewidth=2)  # 绿色边框
plt.fill(angles,Python,facecolor='g',alpha=0.2)  #绿色背景填充
plt.thetagrids(angles*180/np.pi,labels)  # 标签
plt.show()

四、散布矩阵图

pd_data=pd.read_csv("iris.data")
plt.figure()

实验

# (1)分别绘制 OXY 与 time 和 age 的散布图
from pandas.plotting import scatter_matrix

data = pd.read_table('peo.txt', sep='\s+')
da_ti_ag = data.loc[:,["time","age","OXY"]]
plt.figure()
scatter_matrix(da_ti_ag,figsize=(10,10)) 
plt.show()

# (2) 1,2,21,22 的 4 个人的调和曲线图(放在同一张图上)

from pandas.plotting import andrews_curves

matplotlib.rcParams['axes.unicode_minus'] =False  # 显示负号
data = pd.read_table('peo.txt', sep='\s+')
data = data.iloc[:,1:]
da_ti_ag = data.iloc[[1,2,21,22],:]
plt.figure()
andrews_curves(da_ti_ag,"OXY")
plt.show()
